# OpenClaw Workspace

Personal AI assistant workspace for Dandy233.

## Structure

- `AGENTS.md` - Agent configuration and rules
- `BOOTSTRAP.md` - Initialization guide
- `HEARTBEAT.md` - Periodic check tasks
- `IDENTITY.md` - AI identity configuration
- `MEMORY.md` - Long-term memory storage
- `SOUL.md` - Personality and behavior guidelines
- `TOOLS.md` - Tool usage notes
- `USER.md` - User preferences and info
- `diary/` - Daily diary entries
- `memory/` - Daily memory logs
- `scripts/` - Automation scripts

## Scripts

- `scripts/health-check.sh` - System health monitoring
- `scripts/heartbeat-check.sh` - Heartbeat checks
- `scripts/review-memory.sh` - Weekly memory review
- `scripts/auto-sync.sh` - Git auto-sync

## Cron Jobs

- `daily-it-news` - Daily IT news briefing (9:00 AM)
- `weekly-system-check` - Weekly system health check (Mon 10:00 AM)
- `weekly-memory-review` - Weekly memory review (Sun 22:00)
- `auto-git-sync` - Hourly git sync
- `daily-feishu-summary` - Daily Feishu conversation summary
- `weekly-feishu-summary` - Weekly Feishu summary
- `monthly-feishu-summary` - Monthly Feishu summary
- `daily-cleanup` - Daily cleanup tasks

## Usage

This workspace is managed by OpenClaw AI assistant for task automation, memory management, and system monitoring.
