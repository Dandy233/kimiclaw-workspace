## 二、展示方式设计

### 2.1 表格结构优化

| 列 | 内容 | 设计说明 |
|----|------|---------|
| 看过 | [ ] 复选框 | 保留原有标记功能 |
| 作品 | 名称+豆瓣链接 | 快速跳转查看海报/评分 |
| 年份 | 播出年份 | 简洁显示 |
| 制作信息 | 公司/导演 | 核心信息前置 |
| 集数/时长 | 📺291集 | 图标+数字，节省空间 |
| 平台 | 🅱️👑 | 图标+悬停提示 |
| 经典度 | ⭐⭐⭐⭐⭐ | 原作评分保留 |
| OST | ⭐⭐⭐⭐⭐ | OST推荐等级 |
| ▶ | [▶] | 展开详情按钮 |

### 2.2 信息密度控制

**解决方案**：
1. **图标替代文字**：使用emoji图标代替平台名称（🅱️=B站、✨=Disney+）
2. **可折叠详情**：点击 ▶ 展开完整信息（导演详情、完整OST列表等）
3. **分层展示**：
   - 表格行：核心信息（制作公司、集数、平台）
   - 折叠区：详细信息（完整OST、主题曲列表、收藏建议）

### 2.3 图标系统

```
媒体类型:
📺 = TV动画
🎬 = 电影
💿 = OVA
🔴 = 连载中

平台:
🅱️ = B站
📺 = 优酷
🥝 = 爱奇艺
🎬 = Netflix
✨ = Disney+
📁 = 网盘/其他

付费标识:
🆓 = 免费
👑 = 会员
💰 = 付费

OST等级:
⭐⭐⭐⭐⭐ = 必收神作
⭐⭐⭐⭐ = 强烈推荐
⭐⭐⭐ = 值得收藏
```

### 3.2 表格宽度控制 ✅

**原问题**：添加新列后表格过宽

**解决方案**：
1. ✅ 使用emoji图标代替文字（节省50%+空间）
2. ✅ 制作信息列使用"公司/导演"简洁格式
3. ✅ 平台列使用图标组合
4. ✅ 详情内容移至折叠区域

**优化前**：
```
| 看过 | 作品 | 年份 | 简介 | 经典度 | 导演 | 制作公司 | 集数 | B站 | 优酷 | OST作曲家 | 主题曲 | 收藏建议 |
```
→ 11列，过宽

**优化后**：
```
| 看过 | 作品 | 年份 | 制作信息 | 集数 | 平台 | 经典度 | OST | ▶ |
```
→ 9列，其中3列为图标，紧凑
